ariel is a zero-config CLI tool for animating Mermaid diagrams as images and videos.

## Why ariel

- **Fast** — no setup, just install and run
- **Scriptable** — every command supports `--json` for pipeline use
- **Safe** — `--dry-run` previews changes before applying them
